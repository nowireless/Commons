# Commons
This is a collection of useful java components that are useful or neat.
## How to build

### Requirements
 * Maven 3 is required to build.

### Prepare Dependencies

#### MStore
 1. Clone `https://github.com/nowireless/MStore.git` somewhere.
 2. `cd MStore`
 3. `mvn clean install`

### Building
 * `mvn clean package`
